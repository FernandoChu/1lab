Recall that we defined a cartesian category to be one which admits all binary products, hence products of any finite positive cardinality. Such a category is called cartesian closed (abbreviation: ccc) if it has a terminal object (hence products of any natural number of objects), and, for any object AAA, the functor ‚àí√óA- \times A‚àí√óA has a right adjoint, to be denoted [A,‚àí][A,-][A,‚àí].

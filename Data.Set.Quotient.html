Set quotients🔗

Let AAA be a type (more commonly, this will be a set) and R:A→A→TypeR : A \to A \to \tyR:A→A→Type be an equivalence relation valued in propositions. The quotient set A/RA/RA/R is intuitively the set obtained by “gluing together” the elements of AAA which are related by RRR . This can be described more precisely in terms of a universal property: The quotient A/RA/RA/R is the coequalizer of the diagram

Using a higher inductive type, we can define the quotient A/RA/RA/R as the type generated by a map inc:A→A/R\mathrm{inc} : A \to A/Rinc:A→A/R , and identifying the images of related objects.

data _/_ (A : Type ℓ) (R : A → A → Type ℓ') : Type (ℓ ⊔ ℓ') where
  inc    : A → A / R
  quot   : ∀ x y → R x y → inc x ≡ inc y

However, since higher inductive types are homotopy colimits, this does not have the right h-level in general: The quotient 1/⊤1/\top1/⊤ of the unit type by the trivial relation ends up being the circle, and not the unit type. To address this, we throw in a constructor squash, which asserts that the quotient is a set.

  squash : isSet (A / R)

The elimination rule for set-quotients says that you can make (dependent) maps A/R→BA / R \to BA/R→B out of (dependent) maps f:A→Bf : A \to Bf:A→B , as long as BBB is a (family of) set(s), and fff respects the equivalence relation.

Quot-elim : ∀ {ℓ} {B : A / R → Type ℓ}
          → (∀ x → isSet (B x))
          → (f : ∀ x → B (inc x))
          → (∀ x y (r : R x y) → PathP (λ i → B (quot x y r i)) (f x) (f y))
          → ∀ x → B x
Quot-elim Bs bi br (inc x) = bi x
Quot-elim Bs bi br (quot x y r i) = br x y r i
Quot-elim Bs bi br (squash x y p q i j) =
  isHLevel→isHLevelDep 1 Bs 
    (g x) (g y) (λ i → g (p i)) (λ i → g (q i)) (squash x y p q) i j
  where g = Quot-elim Bs bi br

Effectivity🔗

The most well-behaved case of quotients is when R:A→A→TypeR : A \to A \to \tyR:A→A→Type takes values in propositions, is reflexive, transitive and symmetric (an equivalence relation). In this case, we have that the quotient A/RA / RA/R is effective: The map quot is an equivalence.

module _ {A : Type ℓ} {R : A → A → Type ℓ'}
         (Rp : ∀ x y → isProp (R x y))
         (rr : ∀ {x} → R x x)
         (rt : ∀ {x y z} → R x y → R y z → R x z)
         (rs : ∀ {x y} → R x y → R y x)
  where

We will show this using an encode-decode method. For each x:Ax : Ax:A , we define a type family Codex(p)\mathrm{Code}_x(p)Codex​(p) , which represents an equality inc(x)=y\mathrm{inc}(x) = yinc(x)=y . Importantly, the fibre over inc(y)\mathrm{inc}(y)inc(y) will be R(x,y)R(x, y)R(x,y) , so that the existence of functions converting between Codex(y)\mathrm{Code}_x(y)Codex​(y) and paths inc(x)=y\mathrm{inc}(x) = yinc(x)=y is enough to establish effectivity of the quotient.

  private
    Code : A → A / R → Prop ℓ'
    Code x = Quot-elim 
      (λ x → isHLevel-nType 1) 
      (λ y → {- 1 -} R x y , Rp x y)
      λ y z r → 
        Σ≡Prop (λ _ → isProp-isProp) 
          (ua {- 2 -} (propExt (Rp _ _) (Rp _ _) (λ z → rt z r) λ z → rt z (rs r)))

We do quotient induction into the type of propositions, which by univalence is a set. Since the fibre over inc(y)\mathrm{inc}(y)inc(y) must be R(x,y)R(x, y)R(x,y) , that’s what we give for the inc constructor ({- 1 -}, above). For this to respect the quotient, it suffices to show that, given R(y,z)R(y,z)R(y,z) , we have R(x,y)⇔R(x,z)R(x,y) \Leftrightarrow R(x,z)R(x,y)⇔R(x,z) , which follows from the assumption that RRR is an equivalence relation ({- 2 -}).

    encode : ∀ x y (p : inc x ≡ y) → Code x y .fst
    encode x y p = subst (λ y → Code x y .fst) p rr

    decode : ∀ x y (p : Code x y .fst) → inc x ≡ y
    decode x y p = 
      Quot-elimProp {B = λ y → (p : Code x y .fst) → inc x ≡ y} 
        (λ _ → isHLevelΠ 1 λ _ → squash _ _) (λ y r → quot _ _ r) y p

For encode, it suffices to transport the proof that RRR is reflexive along the given proof, and for decoding, we eliminate from the quotient to a proposition. It boils down to establishing that R(x,y)→inc(x)≡inc(y)R(x,y) \to \mathrm{inc}(x) \equiv \mathrm{inc}(y)R(x,y)→inc(x)≡inc(y) , which is what the constructor quot says. Putting this all together, we get a proof that equivalence relations are effective.

  effective : ∀ {x y : A} → isEquiv (quot {R = R} x y)
  effective {x = x} {y} = 
    propExt (Rp x y) (squash _ _) (decode x (inc y)) (encode x (inc y)) .snd

Universal property🔗

As mentioned above, we expect that the inclusion A→A/RA \to A/RA→A/R should coequalise the two projections from the total space of RRR to AAA . Instead of actually defining what it means for a map to coequalise a diagram of sets, we will unfurl this universal property to an equivalent statement: For a set BBB , the space of maps A/R→BA/R \to BA/R→B is equivalent to the maps A→BA \to BA→B which respect RRR .

Quot-univ : isSet B → (A / R → B) ≃ (Σ[ f ∈ (A → B) ] (∀ x y → R x y → f x ≡ f y))
Quot-univ Bset = Iso→Equiv isom where
  open isIso

  isom : Iso _ _
  isom .fst f = f ∘ inc , λ x y r i → f (quot x y r i)
  isom .snd .inv (f , r) = Quot-rec Bset f r
  isom .snd .rinv _ = refl
  isom .snd .linv f = funext (Quot-elimProp (λ _ → Bset _ _) λ x i → f (inc x))

Images🔗

Let f:A→Bf : A \to Bf:A→B be an ordinary function between sets (or, indeed, arbitrary types). Its image im⁡f\im fimf can be computed as the subset {b∈B:(∃a) b=f(a)}\{ b \in B : (\exists a)\ b = f(a) \}{b∈B:(∃a) b=f(a)}, but this description does not carry over to more general categories: More abstractly, we can say that the image embeds into BBB, and admits a map from AAA (in material set theory, this is fff itself — structurally, it is called the corestriction of fff). Furthermore, these two maps factor fff, in that:

(A→eim⁡f↪mB)=(A→fB) (A \xto{e} \im f \xmono{m} B) = (A \xto{f} B) (Ae​imfm​B)=(Af​B)

While these are indeed two necessary properties of an image, they fail to accurately represent the set-theoretic construction: Observe that, e.g. for f(x)=2xf(x) = 2xf(x)=2x, we could take im⁡f=N\im f = \mathbb{N}imf=N, taking e=fe = fe=f itself and m=N↪idNm = \mathbb{N} \xmono{\id{id}} \mathbb{N}m=Nid​N. This factoring clearly recovers fff, as id∘f=f\id{id} \circ f = fid∘f=f. But by taking this as the image, we’ve lost the information that fff lands in the evens!

We can refine the abstract definition by saying that, for a mono im⁡f↪mB\im f \xmono{m} Bimfm​B to be the image of fff, it must be the smallest subobject of BBB through which fff factors — given any other factoring of f=m′∘e′f = m' \circ e'f=m′∘e′, we must have m⊆m′m \sube m'm⊆m′ in the proset of subobjects of BBB, i.e. there exists some kkk such that m=m′∘km = m' \circ km=m′∘k.

In general categories, monomorphisms of C\ca{C}C may be the wrong notion of “subobject” to use. For example, in topology, we’d rather talk about the image which admits a subspace inclusion onto BBB. We may expand the definition above to work for an arbitrary subclass M⊆Mono(C)M \sube \id{Mono}(\ca{C})M⊆Mono(C) of the monomorphisms of CCC, by requiring that the MMM-image of fff be the smallest MMM-subobject through which fff factors.

Since keeping track of all the factorisations by hand would be fiddly, we formalise the idea of image here using comma categories, namely the idea of universal morphisms as in the construction of adjoints. Fix a morphism f:a→bf : a \to bf:a→b, and consider it as an object of the slice category C/b\ca{C}/bC/b.

For a given subclass of monomorphisms MMM, there is a full subcategory of C/b\ca{C}/bC/b spanned by those maps in MMM — let us call it M/bM/bM/b — admitting an evident ff inclusion F:M/b↪C/bF : M/b \mono \ca{C}/bF:M/b↪C/b. An MMM-image of fff is a universal morphism from fff to FFF.

Class-of-monos : ∀ ℓ → Type _
Class-of-monos ℓ =
  Σ[ M ∈ (∀ {a b} → Hom a b → Type ℓ) ]
    (∀ {a b} {f : Hom a b} → M f → is-monic f)

M-image : ∀ {a b} → Class-of-monos ℓ′ → Hom a b → Type _
M-image {a = a} {b} M f = Universal-morphism (cut f)
  (Forget-full-subcat
    {C = Slice C b}
    {P = (λ o → M .fst (o .map))})

Friendly interface🔗

Since this definition is incredibly abstract and indirect, we provide a very thin wrapper module over M-image which unpacks the definition into friendlier terms.

module M-Image {a b} {M : Class-of-monos ℓ′} {f : Hom a b} (im : M-image M f) where

The first thing to notice is that, being an initial object in the comma category f↙Ff \swarrow Ff↙F, we have an object (c,c↪mb)(c, c \xmono{m} b)(c,cm​b) — ccc is the image object, and mmm is the inclusion map:

  Im : Ob
  Im = im .bot .y .fst .domain

  Im→codomain : Hom Im b
  Im→codomain = im .bot .y .fst .map

Furthermore, this map is both an inclusion (since MMM is a class of monomorphisms), and an MMM-inclusion at that:

  Im→codomain-is-M : M .fst Im→codomain
  Im→codomain-is-M = im .bot .y .snd

  Im→codomain-is-monic : is-monic Im→codomain
  Im→codomain-is-monic = M .snd Im→codomain-is-M

So far, we’ve been looking at the “codomain” part of the object in the comma category. We also have the “morphism” part, which provides our (universal) factoring of fff:

  corestrict : Hom a Im
  corestrict = im .bot .map .map

  image-factors : Im→codomain ∘ corestrict ≡ f
  image-factors = im .bot .map .commutes

This is also the smallest factorisation, which takes quite a lot of data to express. Let’s see it:

Suppose we have

  • Some other object ccc; and,
  • An MMM-monomorphism c↪mbc \xmono{m} bcm​b; and,
  • A corestriction map a→ica \xto{i} cai​c; such that
  • m∘i=fm \circ i = fm∘i=f.

Then we have a map k:im⁡f→ck : \im f \to ck:imf→c (written im≤other-image in the code below), and the canonical inclusion im⁡f↪B\im f \mono Bimf↪B factors through kkk:

  im≤other-image
    : ∀ {c} (m : Hom c b) (M-m : M .fst m) (i : Hom a c)
    → m ∘ i ≡ f
    → Hom Im c
  im≤other-image m M i p = im .has⊥ obj .centre .β .map where
    obj : ↓Obj _ _
    obj .x = tt
    obj .y = cut m , M
    obj .map = record { map = i ; commutes = p }

  im≤other-image-factors
    : ∀ {c} {m : Hom c b} {M : M .fst m} {i : Hom a c}
    → {p : m ∘ i ≡ f}
    → m ∘ im≤other-image m M i p ≡ Im→codomain
  im≤other-image-factors {m = m} {M} {i} {p} = im .has⊥ _ .centre .β .commutes
